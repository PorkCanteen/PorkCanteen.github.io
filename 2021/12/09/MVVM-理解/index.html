MVVM ：Model-View-ViewModel

是一种软件架构模式

  • View
    视图 - 用户在屏幕上看到的结构、布局、外观
  • Model
    模型 - 代表内容的数据访问层
  • ViewModel
    视图模型 - 暴露公共属性和命令的视图的抽象

Vue 中的 MVVM

View 和 Model 没有直接联系，而是通过 ViewModel 进行交互
核心：ViewModel，是一个 Vue 实例
从 View 向 Model 方向，通过 DOM Listeners 检测页面的元素变化并更改 Model 中数据；
从 Model 向 View 方向，当更新 Model 中数据时，通过 Data Bindings 更新页面元素
